# 目的
为了在汇编、vmp中更好的识别各种加密算法, 在这个项目中, 收集c实现一些标准加密, 并且为每个加密写一个README, 用自己的话讲一遍源码和加密流程, 然后概括其的特点, 以应对魔
改加密。

## 每个加密配上流程笔记，以应对魔改的算法。
